🎯 Purpose of This Bridge Track
This 4-week track exists for one reason: some cohorts (e.g. SMK-level vocational students) don't yet have the ICE-diagnostic-technician baseline that Week 1 of the main 12-week EV Diagnosis & Repair program assumes participants already have.
This track builds that baseline explicitly, so Week 1 of the main program isn't anyone's first exposure to systems thinking, shop safety, Ohm's Law, or basic diagnostic reasoning.
🧩 6 Interactive Simulator Types (by Session)
Sequencing — complete a procedure/path by clicking steps in order (e.g. multimeter setup order)
Item selection — identify the correct item(s) among several by clicking (e.g. shop safety violations)
Matching — pair concepts with their descriptions (e.g. DTC prefix letters)
Sorting — classify situations into buckets (e.g. sensor vs. actuator)
Meter reading — read a measured value and judge whether it's normal (e.g. 12V battery voltage)
Branching scenario — a roleplay where outcomes diverge based on your choices
Each session is designed around whichever type best fits that day's learning goal — the same 6-type engine used in the main 12-week EV program, so the format will already feel familiar once participants move on.

✅ Bridge Track Completion Criteria (as of Week B4, Day 5)
Ready to begin Week 1 of the main EV program when:
① Can identify which vehicle system a symptom belongs to, and follows general shop safety habits ② Can apply the 4-step diagnostic pattern (symptom → cause → test → verify) and safely use a multimeter ③ Understands the sensor → ECU → actuator relationship and basic DTC concepts ④ Can document findings precisely and explain them in plain language to a customer ⑤ Honestly self-assesses as ready, or knows exactly which week to repeat

This track does not itself teach anything EV- or high-voltage-specific — that begins in Week 1 of the main 12-week program, once this foundation is confirmed solid.
